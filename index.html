Inércia Elástica (Bouncing)

                                                
Gera aquele balanço de mola perfeito após qualquer keyframe.

                                            

                                            
                                        

                                        
amp = .1; freq = 2.0; decay = 2.0;
n = 0;
if (numKeys > 0) {
  n = nearestKey(time).index;
  if (key(n).time > time) n--;
}
if (n == 0) { t = 0; } else { t = time - key(n).time; }
if (n > 0 && t < 1) {
  v = velocityAtTime(key(n).time - thisComp.frameDuration/10);
  value + v*amp*Math.sin(freq*t*2*Math.PI)/Math.exp(decay*t);
} else { value; }

                                    


                                    

                                        

                                            

                                                
Contador de Dinheiro (Moeda)

                                                
Formata números com pontos e vírgulas (ex: R$ 1.200,50).

                                            

                                            
                                        

                                        
val = effect("Slider Control")("Slider");
num = val.value.toFixed(2).toString().replace(".", ",");
"R$ " + num.replace(/\B(?=(\d{3})+(?!\d))/g, ".");

Auto Fade In / Fade Out

                                                
Faz a opacidade surgir e sumir automaticamente no corte da layer.

                                            

                                            
                                        

                                        
fadeFrames = 15;
fadeTime = fadeFrames * thisComp.frameDuration;
Math.min(linear(time, inPoint, inPoint + fadeTime, 0, 100), linear(time, outPoint - fadeTime, outPoint, 100, 0));
